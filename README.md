To build a predictive machine learning model using supervised learning algorithms.

## Algorithms Used
- Decision Tree
- Random Forest

## Dataset

## Features
- Data preprocessing
- Model training and testing
- Accuracy evaluation
- Confusion Matrix
- ROC Curve visualization

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Outcome
Learned supervised learning and model evaluation techniques using machine learning.
